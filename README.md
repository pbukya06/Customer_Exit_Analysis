**📊 Customer Churn Analysis – Power BI Project**
**🔍 Problem**

Customer churn is a major challenge in the banking sector, as losing customers directly impacts revenue and growth.
This project aims to:

-Identify key factors contributing to customer churn
-Analyze customer behavior patterns
-Provide insights to improve customer retention strategies

**📁 Data**
The dataset contains ~10,000 banking customers with the following features:

-Demographics: Age, Gender, Geography
-Account Details: Balance, Tenure, Number of Products
-Customer Activity: Active/Inactive status
-Financial Info: Credit Card ownership, Estimated Salary
-Target Variable: Churn Status (Exited / Retained)

**⚙️ What I Did**
**🧹 Data Preparation**
-Cleaned and transformed raw data using Power Query
-Handled missing values and inconsistencies
-Standardized formats for accurate analysis
**🏗️ Data Modeling**
-Built relationships using CustomerId
-Created a structured data model for efficient reporting
**🔄 Feature Engineering**
-Created derived columns:
  -Age Groups
  -Customer Segments
-Applied business logic for better segmentation
**📊 Dashboard Development**
-Built an interactive Power BI dashboard, including:
  -Customer Overview
  -Financial Analysis
  -Churn Analysis
-Used DAX to calculate KPIs like churn rate, retention rate

**📈 What I Found (Key Insights)**
-**Churn Rate:** 20.4% (high impact on business)
-**Inactive Customers:** 48.5% → major high-risk segment
-Customers aged **45–54 have the highest churn (~48%)**
-**New customers (0–1 year tenure)** are more likely to leave
-Customers with **multiple products show unexpected churn behavior**
-Inactive users are significantly more likely to churn

**💡 Why It Matters (Business Impact)**
These insights help businesses:
-🎯 Target high-risk customers (inactive + mid-age group)
-🔁 Improve onboarding experience for new customers
-💳 Re-evaluate product bundling strategies
-📢 Design personalized retention campaigns

**👉 Result: Better customer retention and improved revenue stability**
