**📊 Customer Churn Analysis – Power BI Project**
**🔍 Overview**

This project analyzes customer behavior in a retail banking dataset to identify patterns that lead to customer churn. The objective is to generate actionable insights that help improve customer retention and support data-driven business decisions.

The entire workflow was implemented using Power BI, including data cleaning, transformation, data modeling, and dashboard creation.

**📁 Dataset**
Banking customer dataset containing ~10,000 records
**Key features include:**
Customer demographics (Age, Gender, Geography)
Account details (Balance, Tenure, Products)
Activity status (Active/Inactive)
Credit card ownership
Customer churn status (Exited)
**🛠️ Tools & Technologies**
**Power BI Desktop** – Data visualization and dashboard creation
**Power Query** – Data cleaning and transformation
**DAX (Data Analysis Expressions)** – KPI calculations and measures
**Gamma **– Presentation (PPT) creation
**⚙️ Project Steps**
**1. Data Cleaning**
Loaded the raw dataset into Power BI
Cleaned data using Power Query
Handled missing values
Standardized data formats
Removed inconsistencies
**2. Data Modeling**
Established relationships between tables using CustomerId
Created a structured data model for analysis
**3. Data Transformation**
Created derived columns such as:
Age Groups
Customer Segments
Applied basic business rules in Power Query
**4. Dashboard Development**
Built an interactive Power BI dashboard with multiple views:
Customer Overview
Financial Analysis
Customer Exit Analysis
**5. Reporting & Presentation**
Generated a detailed business analysis report (PDF)
Created a presentation (PPT) using Gamma to summarize insights

**📈 Dashboard**
The dashboard provides:

Customer distribution by geography and gender
Active vs inactive customer analysis
Financial insights (Balance, Salary, Product usage)
Churn analysis by age, tenure, and products
Interactive filters for dynamic exploration

(Dashboard screenshots are included in this repository)

**📊 Results & Key Insights**
Total Customers: 9,998
Churn Rate: 20.4%
Retention Rate: 79.6%
Inactive Customers: 48.5% (high-risk segment)
Customers aged 45–54 show the highest churn (~48%)
New customers (0–1 year tenure) have higher exit rates
Customers with more products show unexpected churn trends
Inactive customers are significantly more likely to churn

Download the .pbix file from this repository

Interact with the dashboard using filters and slicers
Review insights and business recommendations
